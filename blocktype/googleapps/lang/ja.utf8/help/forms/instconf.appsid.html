埋め込み方法

Googleドキュメント

教育領域向けGoogleアプリ: 個々のファイルの共有は一般的なGoogleアプリアカウントと同じです。

インスティテューション内のみであなたのファイルを共有できるようインスティテューション管理者がGoogleアプリを設定できます。
この場合、閲覧者にはGoogleアプリログイン画面が表示されGoogleアプリへのログイン後に埋め込みファイルを閲覧できるようになります。

テキストドキュメント

  1. Googleドキュメントのテキストドキュメントを開いてください。
  2. ページ右上にある「共有」をクリックして 「ウェブ上に一般公開」を選択してください。
  3. 「一般公開開始」ボタンをクリックしてください。
  4. 「ドキュメントリンク」または「埋め込みコード」をコピーしてください。
  5. コピーしたものをGoogleアプリブロックに貼り付けてください。

スプレッドシート

  1. Googleドキュメントのスプレッドシートを開いてください。
  2. ページ右上にある「共有」をクリックして 「ウェブ上に一般公開」を選択してください。
  3. 「一般公開開始」ボタンをクリックしてください。
  4. 「ドキュメントリンク」または「埋め込みコード」をコピーしてください。
  5. コピーしたものをGoogleアプリブロックに貼り付けてください。

プレゼンテーション

  1. Googleドキュメントのプレゼンテーションを開いてください。
  2. ページ右上にある「共有」をクリックして 「ウェブ上に一般公開」を選択してください。
  3. 「一般公開開始」ボタンをクリックしてください。
  4. 「ドキュメントリンク」または「埋め込みコード」をコピーしてください。
  5. コピーしたものをGoogleアプリブロックに貼り付けてください。

コレクション

  1. Googleドキュメント内のページ左側にある「マイコレクション」または「共有コレクション」をクリックしてください。
  2. あなたが埋め込みたいコレクションをクリックすることで選択してください。
  3. ページ右上にある「共有」をクリックして「共有設定」を選択してください。
  4. 権限を「ウェブ上で一般公開」に変更した後、変更内容を保存してください。
  5. テキストボックスからリンクをコピーしてください。
  6. コピーしたものをGoogleアプリブロックに貼り付けてください。

Googleドキュメントビューア

この機能はGmailのプレビューのように動作します。インターネットより直接アクセスできるようあなたはファイルを埋め込むことができます。あなたはファイルにアクセスできる (ファイル名を含む) 完全なURLをGoogleアプリブロックに貼り付ける必要があります。

現在、Googleドキュメントビューアは次のファイルタイプ埋め込みのみサポートしています: PDF, TIFF, PPT, DOC, DOCX

注意: あなたがMaharaにアップロードしたファイルには直接アクセスできないためこれらのファイルを埋め込むことはできません。

Googleカレンダー

  1. Googleカレンダーを開いてください。
  2. ページ右上にある「設定」をクリックして「カレンダー設定」を選択してください。
  3. タブメニューの「カレンダー」をクリックしてください。
  4. あなたが埋め込みたいカレンダー名をクリックしてください。
  5. 「カレンダーの情報」ページに「カレンダーを埋め込む」セクションが表示されます。
  6. カレンダーの概観をカスタマイズした後、テキストエリアより埋め込みコードをコピーしてください。
  7. コピーしたものをGoogleアプリブロックに貼り付けてください。

Googleマップ

  1. Googleマップを開いてください。
  2. 左上部にある「マイマップ」リンクをクリックしてください。
  3. あなたが埋め込みたいマップ名をクリックしてください。
  4. ページ右上にある「リンク」リンクをクリックしてください。
  5. マップの概観をカスタマイズした後、テキストエリアより埋め込みコードをコピーしてください。
  6. コピーしたものをGoogleアプリブロックに貼り付けてください。
注意: あなたのGoogleマップを埋め込みたい場合、ステップ2および3に従ってください。あなたが埋め込みたいGoogleマップを制限しない場合、ステップ2および3をスキップしてください。

Googleブックス

  1. Googleブックスを開いてください。
  2. あなたが埋め込みたい書籍を探してください。
  3. 「書籍のプレビュー」リンクまたはボタンをクリックしてください。
  4. ページ右上にある「リンク」リンクをクリックしてください。
  5. 書籍の埋め込みコードをコピーしてください。
  6. コピーしたものをGoogleアプリブロックに貼り付けてください。
