外部ギャラリーURI

外部ギャラリーを埋め込むには、あなたは外部ギャラリーのリンクをコピーした後、テキストエリアに貼り付ける必要があります。

次の外部ギャラリーを埋め込むことができます:

  • Flickr - 例) http://www.flickr.com/photos/12345678@N01/sets/98765432123456789/
  • Picasa - 例) http://picasaweb.google.com/user.name/AlbumName#
  • Panoramio - 例) http://www.panoramio.com/user/123456
  • Photobucket - 例) http://s1234.photobucket.com/albums/ee123/username/ or http://s1234.photobucket.com/albums/ee123/username/AlbumName/
  • Windows Liveフォトギャラリー - 例) http://cid-123456ab7890cd12.photos.live.com/self.aspx/AlbumName/photo.jpg

メモ:

  • Flickrを使用するには、サイト管理者は有効なAPIキーを提供する必要があります。
  • Photobucketを使用するには、サイト管理者は有効なAPIキーおよびシークレットAPIキーを提供する必要があります。
  • Windows Liveフォトギャラリーを正常に動作させるには、アルバム全体を表示するため、アルバム内の写真の1つのリンクをコピーした後、テキストエリアに貼り付ける必要があります。
