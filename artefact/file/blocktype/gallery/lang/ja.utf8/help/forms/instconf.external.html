外部ギャラリーURL

外部ギャラリーを埋め込むにはあなたは外部ギャラリーのリンクをコピーした後、テキストエリアに貼り付ける必要があります。

次の外部ギャラリーを埋め込むことができます:

  • Flickr - 例) https://www.flickr.com/photos/12345678@N01/sets/98765432123456789/
  • Picasa - 例) https://picasaweb.google.com/user.name/AlbumName#
  • Photobucket - 例) https://s1234.photobucket.com/albums/ee123/username/ or https://s1234.photobucket.com/albums/ee123/username/AlbumName/
  • Windows Liveフォトギャラリー - 例) https://cid-123456ab7890cd12.photos.live.com/self.aspx/AlbumName/photo.jpg

メモ:

  • Flickrを使用するにはサイト管理者は有効なAPIキーを提供する必要があります。
  • Photobucketを使用するにはサイト管理者は有効なAPIキーおよびシークレットAPIキーを提供する必要があります。
  • Windows Liveフォトギャラリーを正常に動作させるにはアルバム全体を表示するためアルバム内の写真の1つのリンクをコピーした後、テキストエリアに貼り付ける必要があります。
