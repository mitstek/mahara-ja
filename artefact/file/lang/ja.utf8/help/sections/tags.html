タグ

あなたがMahara内で作成したアーティファクトおよびページにタグ付けすることができます。タグは後であなたのコンテンツを簡単に探すことのできる識別しやすい記述的なラベルです。

検索ボックスに文字を入力することにより、あなたはタグを追加することができます。入力した文字を含むタグがすでに存在する場合、そのタグが表示されるためあなたは選択したりさらに使用することができます。タグが存在しない場合、用語の入力を終えて「Enter」キーを押するとにより、あなたのタグを検索ボックスに追加することができます。

タグの左側にある小さな「x」をクリックすることにより、あなたはタグを削除することができます。

「プロファイル」タグを使用した場合、あなたのプロファイルサイドブロックにアイテムのショートカットとしてタグ付けされたアイテムが表示されます。
