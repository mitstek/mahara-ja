タグ

あなたがMahara内で作成したアーティファクトおよびページにタグ付けできます。タグは後であなたのコンテンツを簡単に探すことのできる識別しやすい記述的なラベルです。

検索ボックスに文字を入力することによりあなたはタグを追加できます。入力した文字を含むタグがすでに存在する場合、そのタグが表示されるためあなたは選択したりさらに使用できます。タグが存在しない場合、用語の入力を終えて「Enter」キーを押すことによりあなたのタグを検索ボックスに追加できます。

タグの左側にある小さな「x」をクリックすることによりあなたはタグを削除できます。

「プロファイル」タグを使用した場合、あなたのプロファイルサイドブロックにアイテムのショートカットとしてタグ付けされたアイテムが表示されます。
