マイファイル

ファイルエリアは、あなたのポートフォリオ内で使用する、フォルダおよびファイルのリポジトリ (保存場所) です。

ファイルエリアにアップロードできるファイル数は、画面右側に表示されるクオータボックスに表示される、あなたに設定されたファイル保存容量の制限に依存します。

あなたは一度にアップロードするため、ダイアログボックスから複数のファイルを選択することができます。

ドラッグ&ドロップにて、あなたのファイルおよびフォルダを整理することができます。

セキュリティ上の理由から、あなたのサイト管理者がいくつかのファイルタイプに関して、システムへのアップロードを制限する場合があります。
